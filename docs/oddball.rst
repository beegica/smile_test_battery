=======
Oddball
=======

.. image:: _static/oddball.png
    :width: 375
    :height: 241
    :align: right

This experiment tests people on their ability to react to rare stimulus verses
reacting to common stimulus. This experiment is usually used with EEG to measure
Event Related Potetials. This experiment makes use of Subroutines, as well as
general Flow and Actions states.
